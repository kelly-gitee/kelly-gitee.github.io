let

  • 有代码块的概念
  • 不允许重复声明
  • 暂时性死区，报错
  • 不支持预解析（变量提升）
  • 防止循环变量变成全局的

const

  • 必须声明和赋值同时进行
  • 简单数据类型，值不能更改
  • 复杂数据类型，地址不能更改

Array 扩展

  • Array.from，类数组转数组，建议都使用这个！
  • Array.prototype.find，找到数组当中第一个符合条件的那一项并返回
  • Array.prototype.findIndex，找到数组当中第一个符合条件的那一项的索引并返回
  • Array.prototype.includes，返回 true or false
