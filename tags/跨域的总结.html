跨域传输数据的几个方法

  1. 使用 jsonp 的方式 ，还记录腾讯天气的案例吗？

  2. 使用 CORS 的方式，这个方案是由服务器端设置的，设置的是 响应中的 响应头

    Access-Control-Allow-Origin: *

    Access-Control-Allow-Methods: get,post,put

    Access-Control-Allow-Headers: token,content-type

    Access-Control-Allow-Credentials: true 设置域了之后，才可以跨域传递cookie

  3. 使用 代理 服务器的方式， 以后会在 webpack 或 nginx 这两个工具用学到

  4. WebSocket 实现通信，一般用在网页聊天的场景下，京东客服

  5. window.postMessage() 在两个窗口间进行传输数据 ，在开发地图相关的应用比较常见

Http 协议中的缓存

  1. 强制缓存 ，强制是什么，就是时间没到，别来找我!

    响应头

    ​ cache-control: public,max-age=3600

    ​ Expires: 过期时间

  2. 协商缓存 ，协商是什么？就是问,打个电话问一下 过期了没，过期了没?

    响应头

    ​ e-tag: xxxx 按文件的内容生成的 一个id

    ​ last-Modify: 时间 最后的修改时间

    请求头

    ​ if-None-Match : 如果不匹配, 304 , 200

    ​ if-Modify-Since: 如果你后来已经修改了文件 ，就给我新的
